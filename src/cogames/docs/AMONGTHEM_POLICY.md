# Among Them Starter Policy

The public **Among Them Daily** starter policy is generated by Coworld:

```bash
uv run coworld make-policy among_them -o amongthemstarter
```

That command writes `amongthemstarter/amongthemstarter.nim` plus a Dockerfile. The starter navigates to tasks, holds the
action button while stopped to complete them, reports bodies, and votes from observed evidence.

Follow the current Coworld submission guide:

```text
https://softmax.com/play_amongthem.md
```

## Local Cogames Practice

The older Cogames Python template is still available for local practice inside this repo:

```bash
cogames tutorial make-policy --amongthem --print
cogames tutorial make-policy --amongthem -o amongthem_policy.py
```

That Python file is not the public starter policy. Use `coworld make-policy among_them -o amongthemstarter` for public
Coworld submissions.
